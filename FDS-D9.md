# FDS Day-9

`Float` `Position` `Flex`

##### Float

- Float 
  - 주로 레이아웃을 구성 할 때 블럭 요소를 가로 정렬하기 위해 사용 되는 중요한 기법.
  
  - float는 요소 위에 떠 있게 하며, 왼쪽이나 오른쪽으로 이동 가능하다.

  - position : absolute  를 사용하면 Float가 고정이 된다.

  - Float의 구성요소 
    
    - none 요소를 떠 있게 하지 않는다. (기본값)

    - right 요소를 오른쪽으로 이동시킨다

    - left 요소를 왼쪽으로 이동시킨다.
